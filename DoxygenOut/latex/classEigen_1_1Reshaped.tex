\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename Xpr\+Type, int Rows = Dynamic, int Cols = Dynamic, int Order = 0$>$\newline
class Eigen\+::\+Reshaped$<$ Xpr\+Type, Rows, Cols, Order $>$}
Expression of a fixed-\/size or dynamic-\/size reshape. 


\begin{DoxyTemplParams}{Template Parameters}
{\em Xpr\+Type} & the type of the expression in which we are taking a reshape \\
\hline
{\em Rows} & the number of rows of the reshape we are taking at compile time (optional) \\
\hline
{\em Cols} & the number of columns of the reshape we are taking at compile time (optional) \\
\hline
{\em Order} & can be Col\+Major or Row\+Major, default is Col\+Major.\\
\hline
\end{DoxyTemplParams}
This class represents an expression of either a fixed-\/size or dynamic-\/size reshape. It is the return type of \doxylink{classEigen_1_1DenseBase_a154c89b42c44e2b07923cf3ba45031aa}{Dense\+Base\+::reshaped(\+NRows\+Type,\+NCols\+Type)} and most of the time this is the only way it is used.

However, in C++98, if you want to directly maniputate reshaped expressions, for instance if you want to write a function returning such an expression, you will need to use this class. In C++11, it is advised to use the {\itshape auto} keyword for such use cases.
