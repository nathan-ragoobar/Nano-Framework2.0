\doxylink{structEigen_1_1TensorSycl_1_1internal_1_1TensorContractionKernel_1_1MemHolder}{Mem\+Holder} this is a place holder struct for creating memory hierarchy in SYCL. Inside SYCL kernel it is not allowed to have dynamic memory allocation. While the local memory is created outside of the kernel and passed to the kernel as an accessor, the private memory can only allowed to be allocated statically.
