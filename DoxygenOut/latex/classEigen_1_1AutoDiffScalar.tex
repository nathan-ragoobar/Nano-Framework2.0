\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename Derivative\+Type$>$\newline
class Eigen\+::\+Auto\+Diff\+Scalar$<$ Derivative\+Type $>$}
A scalar type replacement with automatic differentiation capability. 


\begin{DoxyParams}{Parameters}
{\em Derivative\+Type} & the vector type used to store/represent the derivatives. The base scalar type as well as the number of derivatives to compute are determined from this type. Typical choices include, e.\+g., {\ttfamily Vector4f} for 4 derivatives, or {\ttfamily Vector\+Xf} if the number of derivatives is not known at compile time, and/or, the number of derivatives is large. Note that Derivative\+Type can also be a reference (e.\+g., {\ttfamily Vector\+Xf\&}) to wrap a existing vector into an \doxylink{classEigen_1_1AutoDiffScalar}{Auto\+Diff\+Scalar}. Finally, Derivative\+Type can also be any \doxylink{namespaceEigen}{Eigen} compatible expression.\\
\hline
\end{DoxyParams}
This class represents a scalar value while tracking its respective derivatives using \doxylink{namespaceEigen}{Eigen}\textquotesingle{}s expression template mechanism.

It supports the following list of global math function\+:
\begin{DoxyItemize}
\item std\+::abs, std\+::sqrt, std\+::pow, std\+::exp, std\+::log, std\+::sin, std\+::cos,
\item internal\+::abs, internal\+::sqrt, \doxylink{namespaceEigen_1_1numext_a75dd92ce0095ac2b19578912f874b460}{numext\+::pow}, internal\+::exp, internal\+::log, internal\+::sin, internal\+::cos,
\item internal\+::conj, internal\+::real, internal\+::imag, \doxylink{namespaceEigen_1_1numext_ad59216129a430099e97b4c4b1e2f2984}{numext\+::abs2}.
\end{DoxyItemize}

\doxylink{classEigen_1_1AutoDiffScalar}{Auto\+Diff\+Scalar} can be used as the scalar type of an \doxylink{classEigen_1_1Matrix}{Eigen\+::\+Matrix} object. However, in that case, the expression template mechanism only occurs at the top \doxylink{classEigen_1_1Matrix}{Matrix} level, while derivatives are computed right away.
