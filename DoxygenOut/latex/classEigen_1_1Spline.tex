\subsubsection*{template$<$typename \mbox{\hyperlink{classEigen_1_1Spline_a199bea940fb4dfed5ddfe9519a85d816}{Scalar}}, int Dim, int Degree = Dynamic$>$\newline
class Eigen\+::\+Spline$<$ Scalar, Dim, Degree $>$}
A class representing multi-\/dimensional spline curves. 

The class represents B-\/splines with non-\/uniform knot vectors. Each control point of the B-\/spline is associated with a basis function   \begin{align*}
  C(u) & = \sum_{i=0}^{n}N_{i,p}(u)P_i
\end{align*}
