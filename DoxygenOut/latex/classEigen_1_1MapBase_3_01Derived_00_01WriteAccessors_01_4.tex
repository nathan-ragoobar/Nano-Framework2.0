\subsubsection*{template$<$typename Derived$>$\newline
class Eigen\+::\+Map\+Base$<$ Derived, Write\+Accessors $>$}
Base class for non-\/const dense \doxylink{classEigen_1_1Map}{Map} and \doxylink{classEigen_1_1Block}{Block} expression with direct access. 

This base class provides the non-\/const low-\/level accessors (e.\+g. coeff and coeff\+Ref) of dense \doxylink{classEigen_1_1Map}{Map} and \doxylink{classEigen_1_1Block}{Block} objects with direct access. It inherits \doxylink{classEigen_1_1MapBase_3_01Derived_00_01ReadOnlyAccessors_01_4}{Map\+Base$<$\+Derived, Read\+Only\+Accessors$>$} which defines the const variant for reading specific entries.
