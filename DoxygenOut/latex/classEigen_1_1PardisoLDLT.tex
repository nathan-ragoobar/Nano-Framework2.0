\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename \mbox{\hyperlink{classEigen_1_1PardisoImpl_a2f2436d02c24adffa86fd417ce12b7c3}{Matrix\+Type}}, int Options$>$\newline
class Eigen\+::\+Pardiso\+LDLT$<$ Matrix\+Type, Options $>$}
A sparse direct Cholesky (\doxylink{classEigen_1_1LDLT}{LDLT}) factorization and solver based on the PARDISO library. 

This class allows to solve for A.\+X = B sparse linear problems via a LDL\texorpdfstring{$^\wedge$}{\string^}T Cholesky factorization using the Intel MKL PARDISO library. The sparse matrix A is assumed to be selfajoint and positive definite. For complex matrices, A can also be symmetric only, see the {\itshape Options} template parameter. The vectors or matrices X and B can be either dense or sparse.

By default, it runs in in-\/core mode. To enable PARDISO\textquotesingle{}s out-\/of-\/core feature, set\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{solver.pardisoParameterArray()[59]\ =\ 1;\ }

\end{DoxyCode}



\begin{DoxyTemplParams}{Template Parameters}
{\em Matrix\+Type} & the type of the sparse matrix A, it must be a Sparse\+Matrix$<$$>$ \\
\hline
{\em Options} & can be any bitwise combination of Upper, Lower, and Symmetric. The default is Upper, meaning only the upper triangular part has to be used. Symmetric can be used for symmetric, non-\/selfadjoint complex matrices, the default being to assume a selfadjoint matrix. Upper\texorpdfstring{$\vert$}{|}\+Lower can be used to tell both triangular parts can be used as input.\\
