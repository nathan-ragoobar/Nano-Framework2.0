class Eigen\+::\+Comma\+Initializer$<$ Matrix\+Type $>$}
Helper class used by the comma initializer operator. 

This class is internally used to implement the comma initializer feature. It is the return type of \doxylink{classEigen_1_1DenseBase_ae55fbfa5630cabf2a9cba1248581bc66}{Matrix\+Base\+::operator$<$$<$}, and most of the time this is the only way it is used.
