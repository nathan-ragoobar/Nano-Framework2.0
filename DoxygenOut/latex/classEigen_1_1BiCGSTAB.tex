\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename \+\_\+\+Matrix\+Type, typename \+\_\+\+Preconditioner$>$\newline
class Eigen\+::\+Bi\+CGSTAB$<$ \+\_\+\+Matrix\+Type, \+\_\+\+Preconditioner $>$}
A bi conjugate gradient stabilized solver for sparse square problems. 

This class allows to solve for A.\+x = b sparse linear problems using a bi conjugate gradient stabilized algorithm. The vectors x and b can be either dense or sparse.


\begin{DoxyTemplParams}{Template Parameters}
{\em \+\_\+\+Matrix\+Type} & the type of the sparse matrix A, can be a dense or a sparse matrix. \\
\hline
{\em \+\_\+\+Preconditioner} & the type of the preconditioner. Default is \doxylink{classEigen_1_1DiagonalPreconditioner}{Diagonal\+Preconditioner}\\
\hline
\end{DoxyTemplParams}
\textbackslash{}implsparsesolverconcept

The maximal number of iterations and tolerance value can be controlled via the \doxylink{classEigen_1_1IterativeSolverBase_a1bbaf86b7df6d54e78ced4da6ed2dac1}{set\+Max\+Iterations()} and \doxylink{classEigen_1_1IterativeSolverBase_ac43c50c69a5b621cb5a9991a1005750d}{set\+Tolerance()} methods. The defaults are the size of the problem for the maximal number of iterations and \doxylink{structEigen_1_1GenericNumTraits_a3a573c69713fbb143fb49af7db463cee}{Num\+Traits$<$\+Scalar$>$\+::epsilon()} for the tolerance.

The tolerance corresponds to the relative residual error\+: \texorpdfstring{$\vert$}{|}\+Ax-\/b\texorpdfstring{$\vert$}{|}/\texorpdfstring{$\vert$}{|}b\texorpdfstring{$\vert$}{|}

{\bfseries{Performance\+:}} when using sparse matrices, best performance is achied for a row-\/major sparse matrix format. Moreover, in this case multi-\/threading can be exploited if the user code is compiled with Open\+MP enabled. See Topic\+Multi\+Threading for details.
