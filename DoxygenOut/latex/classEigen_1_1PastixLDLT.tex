\subsubsection*{template$<$typename \mbox{\hyperlink{classEigen_1_1PastixBase_adf9fd907efea31ead0f748adf543eb06}{\+\_\+\+Matrix\+Type}}, int \+\_\+\+Up\+Lo$>$\newline
class Eigen\+::\+Pastix\+LDLT$<$ \+\_\+\+Matrix\+Type, \+\_\+\+Up\+Lo $>$}
A sparse direct supernodal Cholesky (\doxylink{classEigen_1_1LLT}{LLT}) factorization and solver based on the Pa\+StiX library. 

This class is used to solve the linear systems A.\+X = B via a LDL\texorpdfstring{$^\wedge$}{\string^}T supernodal Cholesky factorization available in the Pa\+StiX library. The matrix A should be symmetric and positive definite WARNING Selfadjoint complex matrices are not supported in the current version of Pa\+StiX The vectors or matrices X and B can be either dense or sparse
