\subsubsection*{template$<$typename Matrix\+Type, unsigned int Up\+Lo$>$\newline
class Eigen\+::\+Sparse\+Self\+Adjoint\+View$<$ Matrix\+Type, Up\+Lo $>$}
Pseudo expression to manipulate a triangular sparse matrix as a selfadjoint matrix. 


\begin{DoxyParams}{Parameters}
{\em Matrix\+Type} & the type of the dense matrix storing the coefficients \\
\hline
{\em Mode} & can be either {\ttfamily \doxylink{group__enums_gga39e3366ff5554d731e7dc8bb642f83cda891792b8ed394f7607ab16dd716f60e6}{Lower}} or {\ttfamily \doxylink{group__enums_gga39e3366ff5554d731e7dc8bb642f83cda6bcb58be3b8b8ec84859ce0c5ac0aaec}{Upper}} \\
\hline
\end{DoxyParams}
This class is an expression of a sefladjoint matrix from a triangular part of a matrix with given dense storage of the coefficients. It is the return type of \doxylink{classEigen_1_1MatrixBase_a6b0bb99692bd754f0947e9d633f0a627}{Matrix\+Base\+::selfadjoint\+View()} and most of the time this is the only way that it is used.
