\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename \mbox{\hyperlink{classEigen_1_1LLT_a7c385df0cbd88a010a9edc97d7e8a106}{Matrix\+Type}}, int Up\+Lo = Lower$>$\newline
class Eigen\+::\+LLT$<$ Matrix\+Type, Up\+Lo $>$}
Standard Cholesky decomposition (LL\texorpdfstring{$^\wedge$}{\string^}T) of a matrix and associated features. 


\begin{DoxyTemplParams}{Template Parameters}
{\em \+\_\+\+Matrix\+Type} & the type of the matrix of which we are computing the LL\texorpdfstring{$^\wedge$}{\string^}T Cholesky decomposition \\
\hline
{\em \+\_\+\+Up\+Lo} & the triangular part that will be used for the decompositon\+: Lower (default) or Upper. The other triangular part won\textquotesingle{}t be read.\\
\hline
\end{DoxyTemplParams}
This class performs a LL\texorpdfstring{$^\wedge$}{\string^}T Cholesky decomposition of a symmetric, positive definite matrix A such that A = LL\texorpdfstring{$^\wedge$}{\string^}\texorpdfstring{$\ast$}{*} = U\texorpdfstring{$^\wedge$}{\string^}\texorpdfstring{$\ast$}{*}U, where L is lower triangular.

While the Cholesky decomposition is particularly useful to solve selfadjoint problems like D\texorpdfstring{$^\wedge$}{\string^}\texorpdfstring{$\ast$}{*}D x = b, for that purpose, we recommend the Cholesky decomposition without square root which is more stable and even faster. Nevertheless, this standard Cholesky decomposition remains useful in many other situations like generalised eigen problems with hermitian matrices.

Remember that Cholesky decompositions are not rank-\/revealing. This \doxylink{classEigen_1_1LLT}{LLT} decomposition is only stable on positive definite matrices, use \doxylink{classEigen_1_1LDLT}{LDLT} instead for the semidefinite case. Also, do not use a Cholesky decomposition to determine whether a system of equations has a solution.

{\bfseries{Performance\+:}} for best performance, it is recommended to use a column-\/major storage format with the Lower triangular part (the default), or, equivalently, a row-\/major storage format with the Upper triangular part. Otherwise, you might get a 20\% slowdown for the full factorization step, and rank-\/updates can be up to 3 times slower.

This class supports the \doxylink{}{inplace decomposition } mechanism.

Note that during the decomposition, only the lower (or upper, as defined by \+\_\+\+Up\+Lo) triangular part of A is considered. Therefore, the strict lower part does not have to store correct values.
