\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename \+\_\+\+Scalar$>$\newline
class Eigen\+::\+Diagonal\+Preconditioner$<$ \+\_\+\+Scalar $>$}
A preconditioner based on the digonal entries. 

This class allows to approximately solve for A.\+x = b problems assuming A is a diagonal matrix. In other words, this preconditioner neglects all off diagonal entries and, in \doxylink{namespaceEigen}{Eigen}\textquotesingle{}s language, solves for\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{A.diagonal().asDiagonal()\ .\ \mbox{\hyperlink{namespaceEigen_1_1numext_ac5e96d65e0b577673dd47bfe1068cdd9}{x}}\ =\ \mbox{\hyperlink{abseil-cpp_2absl_2container_2internal_2layout__test_8cc_ad66453096871179e6c6effe0df4b483b}{b}}}

\end{DoxyCode}



\begin{DoxyTemplParams}{Template Parameters}
{\em \+\_\+\+Scalar} & the type of the scalar.\\
\hline
\end{DoxyTemplParams}
\textbackslash{}implsparsesolverconcept

This preconditioner is suitable for both selfadjoint and general problems. The diagonal entries are pre-\/inverted and stored into a dense vector.

\begin{DoxyNote}{Note}
A variant that has yet to be implemented would attempt to preserve the norm of each column.
