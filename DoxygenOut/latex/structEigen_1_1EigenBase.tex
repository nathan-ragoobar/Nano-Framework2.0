\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename Derived$>$\newline
class Eigen\+::\+Eigen\+Base$<$ Derived $>$}
Common base class for all classes T such that \doxylink{classEigen_1_1MatrixBase}{Matrix\+Base} has an operator=(\+T) and a constructor Matrix\+Base(\+T).

In other words, an \doxylink{structEigen_1_1EigenBase}{Eigen\+Base} object is an object that can be copied into a \doxylink{classEigen_1_1MatrixBase}{Matrix\+Base}.

Besides Matrix\+Base-\/derived classes, this also includes special matrix classes such as diagonal matrices, etc.

Notice that this class is trivial, it is only used to disambiguate overloaded functions.
