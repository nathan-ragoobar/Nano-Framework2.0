\subsubsection*{template$<$typename Matrix\+Type$>$\newline
class Eigen\+::\+Transpose$<$ Matrix\+Type $>$}
Expression of the transpose of a matrix. 


\begin{DoxyTemplParams}{Template Parameters}
{\em Matrix\+Type} & the type of the object of which we are taking the transpose\\
\hline
\end{DoxyTemplParams}
This class represents an expression of the transpose of a matrix. It is the return type of \doxylink{classEigen_1_1DenseBase_af891b9782f712151a471b3d7cfaaa2e8}{Matrix\+Base\+::transpose()} and \doxylink{classEigen_1_1MatrixBase_a73d8782980bd75141785340d3a7a8e10}{Matrix\+Base\+::adjoint()} and most of the time this is the only way it is used.
