\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename \mbox{\hyperlink{classEigen_1_1PardisoImpl_a2f2436d02c24adffa86fd417ce12b7c3}{Matrix\+Type}}, int \+\_\+\+Up\+Lo$>$\newline
class Eigen\+::\+Pardiso\+LLT$<$ Matrix\+Type, \+\_\+\+Up\+Lo $>$}
A sparse direct Cholesky (\doxylink{classEigen_1_1LLT}{LLT}) factorization and solver based on the PARDISO library. 

This class allows to solve for A.\+X = B sparse linear problems via a LL\texorpdfstring{$^\wedge$}{\string^}T Cholesky factorization using the Intel MKL PARDISO library. The sparse matrix A must be selfajoint and positive definite. The vectors or matrices X and B can be either dense or sparse.

By default, it runs in in-\/core mode. To enable PARDISO\textquotesingle{}s out-\/of-\/core feature, set\+:

\DoxyCodeLine{solver.pardisoParameterArray()[59]\ =\ 1;\ }

\end{DoxyCode}
