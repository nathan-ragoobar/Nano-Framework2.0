\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename \+\_\+\+Functor, \mbox{\hyperlink{namespaceEigen_a53f19342e6b4ecd38ae002d470299add}{Numerical\+Diff\+Mode}} mode = Forward$>$\newline
class Eigen\+::\+Numerical\+Diff$<$ \+\_\+\+Functor, mode $>$}
This class allows you to add a method \doxylink{classEigen_1_1NumericalDiff_a8fc63f1c3307cc6e61dc4d70c57b5037}{df()} to your functor, which will use numerical differentiation to compute an approximate of the derivative for the functor. Of course, if you have an analytical form for the derivative, you should rather implement \doxylink{classEigen_1_1NumericalDiff_a8fc63f1c3307cc6e61dc4d70c57b5037}{df()} by yourself.

More information on \href{http://en.wikipedia.org/wiki/Numerical_differentiation}{\texttt{ http\+://en.\+wikipedia.\+org/wiki/\+Numerical\+\_\+differentiation}}

Currently only "{}\+Forward"{} and "{}\+Central"{} scheme are implemented.
