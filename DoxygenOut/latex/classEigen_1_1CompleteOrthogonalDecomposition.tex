\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename \+\_\+\+Matrix\+Type$>$\newline
class Eigen\+::\+Complete\+Orthogonal\+Decomposition$<$ \+\_\+\+Matrix\+Type $>$}
Complete orthogonal decomposition (COD) of a matrix. 


\begin{DoxyParams}{Parameters}
{\em Matrix\+Type} & the type of the matrix of which we are computing the COD.\\
\hline
\end{DoxyParams}
This class performs a rank-\/revealing complete orthogonal decomposition of a matrix {\bfseries{A}} into matrices {\bfseries{P}}, {\bfseries{Q}}, {\bfseries{T}}, and {\bfseries{Z}} such that     \[
 \mathbf{A} \, \mathbf{P} = \mathbf{Q} \,
                    \begin{bmatrix} \mathbf{T} &  \mathbf{0} \\
                                    \mathbf{0} & \mathbf{0} \end{bmatrix} \, \mathbf{Z}
\] by using Householder transformations. Here, {\bfseries{P}} is a permutation matrix, {\bfseries{Q}} and {\bfseries{Z}} are unitary matrices and {\bfseries{T}} an upper triangular matrix of size rank-\/by-\/rank. {\bfseries{A}} may be rank deficient.

This class supports the \doxylink{}{inplace decomposition } mechanism.

Allows to prescribe a threshold to be used by certain methods, such as \doxylink{classEigen_1_1CompleteOrthogonalDecomposition_af348f64b26f8467a020062c22b748806}{rank()}, who need to determine when pivots are to be considered nonzero. Most be called before calling \doxylink{classEigen_1_1CompleteOrthogonalDecomposition_a36dcdb91c491eb2a0b2669ea00d2c186}{compute()}.

When it needs to get the threshold value, \doxylink{namespaceEigen}{Eigen} calls \doxylink{classEigen_1_1CompleteOrthogonalDecomposition_a3909f07268496c0f08f1b57331d91075}{threshold()}. By default, this uses a formula to automatically determine a reasonable threshold. Once you have called the present method \doxylink{classEigen_1_1CompleteOrthogonalDecomposition_ad60b1167378ca456fbc80687b6da081e}{set\+Threshold(const Real\+Scalar\&)}, your value is used instead.
