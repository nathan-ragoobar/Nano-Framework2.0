\subsubsection*{template$<$typename \mbox{\hyperlink{classEigen_1_1TriangularView_ad07cddd04018881a8b21f3bb4e34d7d7}{Matrix\+Type}}, unsigned int Mode$>$\newline
class Eigen\+::\+Triangular\+View$<$ Matrix\+Type, Mode $>$}
Expression of a triangular part in a matrix. 


\begin{DoxyParams}{Parameters}
{\em Matrix\+Type} & the type of the object in which we are taking the triangular part \\
\hline
{\em Mode} & the kind of triangular matrix expression to construct. Can be \doxylink{group__enums_gga39e3366ff5554d731e7dc8bb642f83cda6bcb58be3b8b8ec84859ce0c5ac0aaec}{Upper}, \doxylink{group__enums_gga39e3366ff5554d731e7dc8bb642f83cda891792b8ed394f7607ab16dd716f60e6}{Lower}, \doxylink{group__enums_gga39e3366ff5554d731e7dc8bb642f83cda8b0b63ff46dcd7377b59df14e0562fc1}{Unit\+Upper}, \doxylink{group__enums_gga39e3366ff5554d731e7dc8bb642f83cda8155cfdfde9e75e7144dff0393d17181}{Unit\+Lower}, \doxylink{group__enums_gga39e3366ff5554d731e7dc8bb642f83cdae38aad7d66fecfb213fce453edff4c7a}{Strictly\+Upper}, or \doxylink{group__enums_gga39e3366ff5554d731e7dc8bb642f83cda00aa79c8865ada1584c4d57bcd93299f}{Strictly\+Lower}. This is in fact a bit field; it must have either \doxylink{group__enums_gga39e3366ff5554d731e7dc8bb642f83cda6bcb58be3b8b8ec84859ce0c5ac0aaec}{Upper} or \doxylink{group__enums_gga39e3366ff5554d731e7dc8bb642f83cda891792b8ed394f7607ab16dd716f60e6}{Lower}, and additionally it may have \doxylink{group__enums_gga39e3366ff5554d731e7dc8bb642f83cdaddb72f888ac85d5a1c52333e54f9374b}{Unit\+Diag} or \doxylink{group__enums_gga39e3366ff5554d731e7dc8bb642f83cda884ff7240392e85aa6e4b3c957e36483}{Zero\+Diag} or neither.\\
\hline
\end{DoxyParams}
This class represents a triangular part of a matrix, not necessarily square. Strictly speaking, for rectangular matrices one should speak of "{}trapezoid"{} parts. This class is the return type of \doxylink{classEigen_1_1MatrixBase_a766e24ff164cf92d9a43702a7accdda3}{Matrix\+Base\+::triangular\+View()} and \doxylink{classEigen_1_1SparseMatrixBase_a6762232f126791a057dfbf3bf6014542}{Sparse\+Matrix\+Base\+::triangular\+View()}, and most of the time this is the only way it is used.
