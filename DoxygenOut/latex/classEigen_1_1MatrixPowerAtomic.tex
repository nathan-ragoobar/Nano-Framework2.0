\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename Matrix\+Type$>$\newline
class Eigen\+::\+Matrix\+Power\+Atomic$<$ Matrix\+Type $>$}
Class for computing matrix powers. 


\begin{DoxyTemplParams}{Template Parameters}
{\em Matrix\+Type} & type of the base, expected to be an instantiation of the \doxylink{classEigen_1_1Matrix}{Matrix} class template.\\
\hline
\end{DoxyTemplParams}
This class is capable of computing triangular real/complex matrices raised to a power in the interval $ (-1, 1) $.

\begin{DoxyNote}{Note}
Currently this class is only used by \doxylink{classEigen_1_1MatrixPower}{Matrix\+Power}. One may insist that this be nested into \doxylink{classEigen_1_1MatrixPower}{Matrix\+Power}. This class is here to facilitate future development of triangular matrix functions. 
\end{DoxyNote}
