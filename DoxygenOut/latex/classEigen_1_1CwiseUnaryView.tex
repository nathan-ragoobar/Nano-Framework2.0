\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename View\+Op, typename Matrix\+Type$>$\newline
class Eigen\+::\+Cwise\+Unary\+View$<$ View\+Op, Matrix\+Type $>$}
Generic lvalue expression of a coefficient-\/wise unary operator of a matrix or a vector. 


\begin{DoxyTemplParams}{Template Parameters}
{\em View\+Op} & template functor implementing the view \\
\hline
{\em Matrix\+Type} & the type of the matrix we are applying the unary operator\\
\hline
\end{DoxyTemplParams}
This class represents a lvalue expression of a generic unary view operator of a matrix or a vector. It is the return type of \doxylink{namespaceEigen_aa71b5974e117d989a4e79f54ba65f460}{real()} and \doxylink{namespaceEigen_a524f2d0ae40cbbb4f55ea073c98a1c43}{imag()}, and most of the time this is the only way it is used.
