Objects of this type represent the Hessenberg matrix in the Hessenberg decomposition of some matrix. The object holds a reference to the \doxylink{classEigen_1_1HessenbergDecomposition}{Hessenberg\+Decomposition} class until the it is assigned or evaluated for some other reason (the reference should remain valid during the life time of this object). This class is the return type of \doxylink{classEigen_1_1HessenbergDecomposition_a8e781d2e22a2304647bcf0ae913cc8ea}{Hessenberg\+Decomposition\+::matrix\+H()}; there is probably no other use for this class.
