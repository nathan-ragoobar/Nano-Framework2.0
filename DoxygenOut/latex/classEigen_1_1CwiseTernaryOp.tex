\subsubsection*{template$<$typename Ternary\+Op, typename \mbox{\hyperlink{classEigen_1_1CwiseTernaryOp_a742d4a6482bb5cddadca3aa3f75c85ee}{Arg1}}, typename \mbox{\hyperlink{classEigen_1_1CwiseTernaryOp_a138625e6ffb3e5fe57d2a4f8eb251966}{Arg2}}, typename \mbox{\hyperlink{classEigen_1_1CwiseTernaryOp_a514c2697ca522a97b8e5650207bbbed7}{Arg3}}$>$\newline

Generic expression where a coefficient-\/wise ternary operator is applied to two expressions.

This class represents an expression where a coefficient-\/wise ternary operator is applied to three expressions. It is the return type of ternary operators, by which we mean only those ternary operators where all three arguments are \doxylink{namespaceEigen}{Eigen} expressions. For example, the return type of betainc(matrix1, matrix2, matrix3) is a \doxylink{classEigen_1_1CwiseTernaryOp}{Cwise\+Ternary\+Op}.

Most of the time, this is the only way that it is used, so you typically don\textquotesingle{}t have to name \doxylink{classEigen_1_1CwiseTernaryOp}{Cwise\+Ternary\+Op} types explicitly.
